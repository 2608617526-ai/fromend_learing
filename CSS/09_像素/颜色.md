在CSS中，颜色可以通过多种方式表示，常见的有以下几种：

### 1. 预定义颜色名称
CSS 定义了一系列预定义的颜色名称，这些名称直观且易于记忆。例如：
```css
p {
    color: red;
    background - color: lightblue;
}
```
这里 `red`（红色）和 `lightblue`（浅蓝色）就是预定义的颜色名称。常见的预定义颜色名称还有 `black`（黑色）、`white`（白色）、`green`（绿色）、`blue`（蓝色）、`yellow`（黄色）等。虽然方便使用，但可选择的颜色有限。

### 2. 十六进制表示法
十六进制表示法是在CSS中指定颜色最常用的方式之一。它由一个 `#` 符号后跟6位十六进制数字组成，每两位分别代表红色、绿色和蓝色通道的值。例如：
```css
div {
    color: #FF0000; /* 红色 */
    background - color: #00FF00; /* 绿色 */
}
```
十六进制值的范围从 `#000000`（黑色，所有通道值为0）到 `#FFFFFF`（白色，所有通道值为255）。也可以使用3位十六进制表示法，浏览器会自动将其扩展为6位，例如 `#FFF` 等同于 `#FFFFFF`，`#F00` 等同于 `#FF0000`。

### 3. RGB表示法
RGB（红、绿、蓝）表示法通过指定红、绿、蓝三种颜色通道的强度来定义颜色。其语法为 `rgb(红, 绿, 蓝)`，每个值的范围是从0到255。例如：
```css
span {
    color: rgb(255, 0, 0); /* 红色 */
    background - color: rgb(0, 255, 0); /* 绿色 */
}
```
RGB还支持使用百分比来表示颜色通道的强度，例如 `rgb(100%, 0%, 0%)` 也表示红色。

### 4. RGBA表示法
RGBA是RGB的扩展，增加了一个透明度（Alpha）通道。其语法为 `rgba(红, 绿, 蓝, 透明度)`，透明度值的范围是从0（完全透明）到1（完全不透明）。例如：
```css
button {
    background - color: rgba(0, 0, 255, 0.5); /* 半透明蓝色 */
}
```
这使得元素可以呈现出透明或半透明的效果，在创建如悬浮效果、遮罩层等场景中非常有用。

### 5. HSL表示法
HSL（色相、饱和度、亮度）表示法基于人类对颜色的感知来定义颜色。其语法为 `hsl(色相, 饱和度, 亮度)`。
 - **色相（Hue）**：是颜色在色轮上的位置，取值范围是0到360，0（或360）代表红色，120代表绿色，240代表蓝色。
 - **饱和度（Saturation）**：表示颜色的纯度，取值范围是0%（灰色）到100%（完全饱和）。
 - **亮度（Lightness）**：表示颜色的明亮程度，取值范围是0%（黑色）到100%（白色）。例如：
```css
a {
    color: hsl(120, 100%, 50%); /* 纯绿色 */
}
```

### 6. HSLA表示法
HSLA是HSL的扩展，增加了透明度（Alpha）通道，语法为 `hsla(色相, 饱和度, 亮度, 透明度)`，透明度值的范围同样是从0到1。例如：
```css
nav {
    background - color: hsla(240deg, 50%, 50%, 0.8); /* 半透明的蓝色调 */
}
```
HSLA和RGBA类似，都可以创建透明效果，但HSL和HSLA基于人类对颜色的感知，在调整颜色时可能更直观，尤其是在调整颜色的饱和度和亮度方面。
